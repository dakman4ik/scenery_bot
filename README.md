# scenery_bot



ИИ-агент, который по короткому промпту может создать цельный сценарий с описаниями сцен. Также, он подбирает музыку под сцену через неофициальную библиотеку yandex-music. 

Пока проект находится в разработке.
Консольное приложение.

# Быстрый старт
 
Необходимо создать .env файл. Он должен лежать в одной директории с scenerybot.py и должен содержать такие поля:
  * OPENROUTER_API_KEY=ваш_ключ
  * YANDEX_API=оставить_пустым

OPENROUTER_API_KEY - обязателен, получить можно в личном кабинете OpenRouter.

YANDEX_API - Личный токен нужен только для доступа к приватным плейлистам/истории. В этом проекте не используется.

Сборка образа
 * docker build -t scenerybot -f Dockerfile .

Запуск
 * docker run -it --env-file scenerybot/.env scenerybot

Запуск без Docker (сначала надо перейти в директорию проекта через cd)
 * pip install -r requirements.txt
 * python scenerybot.py

# Ограничения
 * OpenRouter ограничил доступ для пользователей из России и Беларуси - запросы с российских IP-адресов блокируются на уровне сервиса (403 Access denied by security policy). Это не баг проекта - для работы из России потребуется VPN.
 * Обработка ошибок API минимальна, проект в разработке.
 * Поиск трека по настроению - эвристический (по текстовому запросу), не гарантирует идеального совпадения.
